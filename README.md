# SellerRadar Free — Meesho Seller Tool

Free, open-source Chrome extension for Meesho sellers. No login, no subscription, no backend required.

## Features

- **Product Info Boxes**: See category, listing price, shipping charge, quality score, orders/day on every product card
- **Product Detail Analytics**: Full review breakdown, rating distribution, daily review chart
- **Review Export**: Download reviews as CSV
- **Image Download**: Download all product images with one click
- **Supplier Dashboard**: "Find on Meesho.com" quick link
- **SPA Aware**: Works seamlessly with Meesho's single-page app navigation

## Installation

1. Download or clone this repository
2. Open Chrome → `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select the extension folder
6. Visit meesho.com — the SellerRadar button appears on the right!

## How It Works

The extension uses Meesho's public product search API to fetch product data and reviews. No external backend or authentication is needed.
